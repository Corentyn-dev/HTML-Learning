Exercices HTML

Exercice 1: Structure d'une page HTML

Créez une structure de base pour une page HTML en utilisant les balises <html>, <head> et <body>.

Exercice 2: Balisage du contenu

Utilisez les balises HTML appropriées pour créer un paragraphe de texte, un titre de niveau 1 et une liste non ordonnée avec au moins trois éléments.

Exercice 3: Insérer une image

Insérez une image dans votre page HTML en utilisant la balise <img>. Assurez-vous que l'image est accessible et qu'elle est bien affichée.

Exercice 4: Créer un lien hypertexte

Ajoutez un lien hypertexte vers une page externe de votre choix en utilisant la balise <a>. Assurez-vous que le lien fonctionne correctement.
